## Project Objective
This capstone presents an end-to-end SAP business scenario for the Data Analytics Engineer track and focuses on:
- Configuring a realistic P2P business flow (MM + FI integration)
- Designing analytics-ready data structures
- Defining KPI logic for management reporting
- Building a dashboard-ready semantic layer for reporting tools

## Selected Project Approach
This project uses:
- End-to-end scenario: Procure-to-Pay (P2P) - Full Purchasing Cycle (SAP MM)
- Financial integration checkpoints from FI for invoice and payment analytics
- Analytics layer and dashboard specification for business decision-making
